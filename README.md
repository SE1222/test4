# Physics Learning Platform

一个集成了Coze AI助手的物理学习平台。

## 功能特点

- ? AI物理助手：通过右下角的悬浮球与AI助手对话
- ??? 物理学家介绍：展示历史上伟大的物理学家
- ? 快速问题：预设的常见物理问题
- ? 响应式设计：支持PC和移动设备

## 部署到GitHub Pages

1. 创建新的GitHub仓库
2. 上传这些文件到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择从main分支部署
5. 获得你的网站链接：`https://yourusername.github.io/repository-name/`

## 使用方法

1. 打开网站
2. 等待AI助手初始化完成（右上角状态显示"AI Assistant Ready"）
3. 点击右下角的悬浮球开始与AI助手对话
4. 或者点击预设问题快速开始

## 技术栈

- HTML5 + CSS3 + JavaScript
- Coze Web SDK
- 响应式设计

## 注意事项

- 每个访问者都有独立的对话会话
- 需要稳定的网络连接
- 支持现代浏览器
